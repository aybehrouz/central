Assume that in an environment, an agent must choose an action from a set of available actions.
The utility of this action for the agent, can be expressed as a real-valued function and depends on the unknown state
of the environment.
More precisely, if we denote the action as $a$ and the state of the environment as $x$, the utility of
each action for the agent is a real-valued function $\util$.
We assume that evidence $e$ of the environment state exists, and our agent can use it to determine the optimal action.

According to Bayesian decision theory, the optimal action when the evidence $e$ is observed, is the action
that maximizes the expected utility conditioned on $e$:

\begin{equation*}
    a_{opt} = \arg\max_a \Ex{\envDist}{\util \mid e} = \arg\max_a\sumx \util \envDist(x \mid e).
\end{equation*}
Here $\envDist(X|E)$ is the probability distribution of the environment state given the evidence.
But how can we get this distribution?
Does there exist some objective distribution in the environment that we need to somehow find out?
From the subjective viewpoint to probability, the answer to this question is negative.
There is not such an objective distribution and $\envDist$ only exists in our mind.
It is a tool that helps us regulate our reasoning, and an intermediate measure which is created inside Bayesian decision
making framework to ensure we stay consistent with common sense principles and our assumptions.

Thus, if we accept the subjective viewpoint, $\envDist$ needs to be determined by one’s prior knowledge and assumptions.
When we do not have any assumptions and prior knowledge, the set of candidate distributions for $\envDist$ is simply
the set of all probability distributions.
As we start making assumptions or using our prior knowledge, the probability distributions that are not consistent
with our assumptions are removed from this set and the set of acceptable probability distributions gets smaller
and smaller.
We shall call this set a probabilistic model:

\begin{definition}[Probabilistic Model]
    We define a \emph{probabilistic model} \M as a pair $\left(\set{S}, \set{P}\right)$, where \set{S} is a sample
    space, and \set{P} is a set of acceptable probability measures on \set{S}.
\end{definition}

One of the most commonly used assumptions in the statistical modeling is the exchangeability assumption.
We say a sequence of random variables $\seq$ is exchangeable if the ordering of variables does not provide any
useful information.
In other words, for an exchangeable sequence of random variables the joint probability is the same
for any permutation of variables and this property holds for every subset of the sequence.
For example, if each $X_i$ is a binary variable, the joint probability depends only on the total number of 1s
and 0s, and the exact ordering of them is irrelevant.

When a sequence of random variables is exchangeable, de Finetti's theorem states that their joint probability has a
special representation:

\begin{equation*}
    P(x_1, x_2, \cdots) = \int \prod_i\post \dt,
\end{equation*}
which means exchangeable observations are conditionally independent relative to some latent parameter $\paramValue$.

Moreover, the theorem states that if each $X_i$ is discrete, $\post$ would be the multinomial distribution.
Therefore, in our decision-making problem, if the evidence $e$ is a sequence of previous experiences of the
environment $\seq$, and if we assume that our environment does not change over time so that this sequence is
exchangeable, then to fully determine $\envDist$, we only need to determine the prior probability of our latent
parameter $\prior$.

Clearly, one way for choosing a good $\prior$ is to use some prior knowledge of the environment.
But what if that knowledge is not available?
Unfortunately, answering to this question is not easy.
We can tell what distributions are not good choices for $\prior$.
For example, any distribution that gives zero probability to some values of $\paramValue$, is not a good choice,
since if we use that prior no evidence can convince us that the value of $\paramValue$ is amongst those values.

However, arguments like this one can only rule out a small fraction of possible distributions for $\prior$, and
we will still have many options that we do not know which one we should choose.
Throughout this writing, we try to develop an approach that helps us choose a good prior distribution when no prior
knowledge of the environment is available.
